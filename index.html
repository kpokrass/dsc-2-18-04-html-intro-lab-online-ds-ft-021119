Steps for Creating HTML Document

  1. Add opening doctype tag and declare code type as html
  2. Add opening and closing html tags to indicate all code within tags should be interpreted as html code
  3. Add opening and closing head tags. The head contains info about the html document and contents for the web browser
  4. Add opening and closing title tags to the head section to give the page a title that will show up in the browser tab
  5. Add opening and closing body tags. The bodt contains the actual content of the web page that a user will interact with
